## 課題訴求

実務で得たスキルがシステムに反映されない。プロジェクト経験が評価・配置に活かされていない。

## ソリューション概要

参加したプロジェクト情報を入力するとスキルタイムラインに自動反映。業務実績・担当ロール・使用技術がそのままスキルデータとして記録される。

## 提供する価値

働くことがそのままスキルの証明になる。実務経験に基づく客観的なスキルデータが自動蓄積され、評価・配置・要員計画の精度が上がる。

<!-- EN -->

## Challenge

Skills gained from actual work aren't reflected in the system. Project experience isn't leveraged for evaluations or placements.

## Solution Overview

Input participated project information and it automatically reflects in the skill timeline. Work achievements, assigned roles, and technologies used are recorded as skill data.

## Value Delivered

Working directly becomes skill proof. Objective skill data based on practical experience auto-accumulates, improving accuracy of evaluations, placements, and workforce planning.
