## 課題訴求

スキル可視化と学習管理が別々のシステムで動いており、スキルギャップの解消に直接繋がらない。

## ソリューション概要

HubのスキルギャップデータがLXPと連動。ギャップ分析結果から学習コンテンツが自動提示され、そのままラーニングパスに追加できる。学習完了後はスキルデータに自動反映。

## 提供する価値

「何を学ぶべきか」をシステムが導いてくれる。スキル可視化から学習開始・スキル更新までシームレスに繋がる育成サイクルを実現する。

<!-- EN -->

## Challenge

Skill visualization and learning management run on separate systems, preventing direct connection to closing skill gaps.

## Solution Overview

Hub's skill gap data integrates with LXP. Learning content is automatically suggested from gap analysis results and can be added directly to learning paths. Skill data auto-updates after learning completion.

## Value Delivered

The system guides you on "what to learn." Achieve a seamless development cycle from skill visualization to learning initiation and skill data updates.
