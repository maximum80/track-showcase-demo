## 課題訴求

プロジェクト完了後のスキルアップデートが手動・属人的。実務で得た経験がスキルデータに反映されず、スキルデータが常に陳腐化している。

## ソリューション概要

プロジェクト完了→担当ロール・業務実績がスキルタイムラインに自動連携。実務で使ったスキル・習熟度がデータとして蓄積される。

## 提供する価値

「働くこと」がそのままスキル証明になる。スキルデータを循環させることで、将来の要員計画精度が向上し、従業員のキャリア開発の質も上がる。

<!-- EN -->

## Challenge

Post-project skill updates are manual and ad hoc. Skills gained from actual work aren't reflected in skill data, causing constant data staleness.

## Solution Overview

Project completion → assigned role and work achievements automatically sync to skill timeline. Skills used and proficiency levels are accumulated as data.

## Value Delivered

"Working" directly becomes skill proof. Circulating skill data improves future workforce planning accuracy and enhances employee career development quality.
