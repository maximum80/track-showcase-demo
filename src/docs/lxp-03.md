## 課題訴求

すでにUdemyやLinkedIn Learningなど外部の学習コンテンツを活用しているが、受講実績がバラバラに分散している。

## ソリューション概要

Udemy等の外部コースをTrack LXPに接続し一元管理。外部コンテンツの受講実績もスキルデータと連携。既存の学習投資を無駄にせずTrackプラットフォームに集約できる。

## 提供する価値

既存の学習投資をTrackに集約できる。社内・社外コンテンツをまたいで統合的な学習管理が実現し、受講者はひとつの画面で全ての学習にアクセスできる。

<!-- EN -->

## Challenge

Already using external learning content like Udemy and LinkedIn Learning, but enrollment records are scattered across platforms.

## Solution Overview

Connect external courses like Udemy to Track LXP for centralized management. External content enrollment records also link to skill data. Consolidate existing learning investments into the Track platform.

## Value Delivered

Consolidate existing learning investments into Track. Achieve integrated learning management across internal and external content, with learners accessing all learning from a single interface.
