## 課題訴求

スキルアセスメントを実施したいが、試験問題の作成に専門知識が必要。問題作成のコストと時間が障壁になっている。

## ソリューション概要

JDやタスクリストを入力するだけでAIが試験問題を自動生成。スキルレベルに応じた難易度調整、選択式・記述式の形式選択も自動で対応。生成後の問題は確認・編集が可能。

## 提供する価値

試験設計の専門知識がなくても高品質な試験が作れる。アセスメント導入の初期コストを大幅削減し、ロール別・スキル別のカスタム試験を短期間で整備できる。

## Challenge

Want to conduct skill assessments but creating exam questions requires specialized knowledge. Question creation cost and time are barriers.

## Solution Overview

Simply input JDs or task lists and AI auto-generates exam questions. Automatic difficulty adjustment by skill level and format selection (multiple choice/written). Generated questions can be reviewed and edited.

## Value Delivered

Create high-quality exams without assessment design expertise. Dramatically reduce initial assessment deployment costs and rapidly prepare custom exams by role and skill.
